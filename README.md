# n8n Workflow Lab

Laboratorio de desarrollo, simulación y pruebas para workflows de n8n.

## Objetivos

- Desarrollar workflows de n8n.
- Crear APIs simuladas.
- Probar integraciones sin afectar producción.
- Versionar workflows en Git.
- Documentar nodos, entradas y salidas.
- Ejecutar pruebas de éxito y error.

## Estructura

- `workflows/development`: workflows en desarrollo.
- `workflows/testing`: workflows aprobados para pruebas.
- `workflows/production`: versiones estables.
- `mock-api`: servicios externos simulados.
- `shared/fixtures`: datos de prueba.
- `tests`: pruebas automatizadas.
- `docs`: documentación técnica.
